Full Stack Web Developer

Having been ill for a long time with anxiety and depression I decided I needed to do something constructive to occupy my time. I first studied for, and graduated with, an honours degree in literature, having had vague ideas about becoming a teacher. After that I came-across the Python programming language and discovered that I love writing code. The opportunity to attend this front end web development bootcamp arose, which I have eagerly taken up. The world wide web is still in its infancy, and I don't want to merely see what happens to it - I really want to be a part of it.

My hopes for the course are that the skills I aquire will be a rock-solid foundation on which to build a career as a web developer. I want to be able to look at a page of code and know exactly what is going on - especially where JavaScript is concerned, because at the moment it still looks pretty much impenetrable. I hope I've made good use of some of the lesson we had on design and I would very much like to build on that aspect of things because I want my front end work to be considered worthy of consideration by potential employers. And if a page doesn't look good then neither do I.
